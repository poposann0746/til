## 目的
- 学習ログを可視化してモチベーションをUPさせる
- 振り返りを通して知識を定着  
- 毎日学習記録をコミットし、確実な習慣づけを行う

---

## 記録フォーマット  

```markdown
# TIL - YYYY/MM/DD

## 学習時間
- 合計: X時間
- 内訳:
  - RUNTEQ教材: Xh
  - 個人開発: Xh
  - その他: Xh

---

## 学習内容
- [ ] 学習したことを箇条書きで
- [ ] 使用した教材・リンク

---

自分なりの要約やTips
